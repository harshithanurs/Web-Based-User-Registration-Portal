# Web-Based User Registration Portal

## Description
The Web-Based User Registration Portal is a PHP and MySQL application that allows users to register and store their details securely. It includes form validation, password encryption, session handling, and an admin module to view registered users.

## Technologies Used
- HTML
- CSS
- PHP
- MySQL

## Features
- User Registration
- Secure Password Storage
- Admin Login and Dashboard
- Country Selection using JSON

## How to Run the Project
1. Install XAMPP
2. Place the project folder in `htdocs`
3. Start Apache and MySQL
4. Import the database in phpMyAdmin
5. Open browser and run `localhost/project-folder-name`
